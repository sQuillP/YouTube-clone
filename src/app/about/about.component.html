What is the YouTube API app?

  • The YouTube API app is exactly as the name entails...It grabs YouTube data from Google's database and provides a watching experience that is similar to the real YouTube.

Optimizing your watching experience

  • The YouTube API has a quota of 10,0000 units however searches cost 100 units per request.

  • To optimize the watching experience, the user can turn on "conserve API" to turn off searches and only fetch videos that have been favorited or previously watched.

  • Retrieving these videos will greatly reduce the quota cost as these videos are stored separately in the RealTime Database provided by Firebase.

  • Using these features require creating an account via email with this application.
